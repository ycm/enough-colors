# enough-colors

Sparse highlighting

## Setup

Inside `.vimrc`:

```vim
set termguicolors
set background=light " or dark
let g:enough_colors_opt_transp_bg=1 " optional, for transparency
colo enough
```

Other configs:

```vim
let g:NERDTreeStatusline = '%#Normal#'
```
